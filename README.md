# tf_voxelnet

An tensorflow inplementation for [voxelnet](https://arxiv.org/abs/1711.06396)

# Usage

1. run `preprocess.py` for pointcloud preprocess(attention to use correct path)
2. run `train.py`
3. launch a tensorboard and wait for it
